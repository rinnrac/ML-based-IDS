This project presents a Machine Learning-Based Intrusion Detection System (IDS) designed to identify and classify malicious network activities in real time. The system utilizes deep learning techniques to analyze network traffic and detect cyber threats such as Denial of Service (DoS) attacks, helping improve network security and threat response capabilities.
Built using the CSE-CIC-IDS2018 dataset, the project performs extensive data preprocessing, feature engineering, and model training to accurately distinguish between benign and malicious traffic. Multiple neural network architectures, including a Feedforward Neural Network (FFNN) and an improved hybrid deep learning model, were implemented and evaluated.
Key Features
  Network traffic analysis and intrusion detection
  Data preprocessing and feature engineering pipeline
  Deep learning-based attack classification
  Detection of DoS attacks such as GoldenEye and Slowloris
  High accuracy, precision, recall, and F1-score
  Model saving and deployment support
  Real-time intrusion detection capability
  Scalable and adaptable cybersecurity solution
Technologies Used
  Python
  TensorFlow / Keras
  NumPy
  Pandas
  Scikit-learn
  Matplotlib
  Deep Learning (FFNN, Hybrid CNN-LSTM Architecture)
Dataset
CSE-CIC-IDS2018 Dataset
